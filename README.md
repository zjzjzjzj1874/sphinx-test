# Sphinx文档

## 基本用法
* 安装
MacOS: `brew install sphinx`
* 创建项目
```
sphinx-quickstart
```
* 构建
```
make html
```
* 打开
```
open build/html/index.html
```

* 创建latexpdf
```
make latexpdf # latexmk这个工具需要哦
```
